## Why it exists

RiffBanks is a collaborative platform for musicians and bands. The idea is to reduce the fragmentation between chats, file sharing, project tracking, and recruitment.

## What it does

The application supports:

- user authentication
- band creation with invite codes
- song workflow management
- audio and image uploads
- real-time chat per song
- voting on assets
- gig postings for recruiting musicians
- mobile-first responsive UI

## Engineering value

RiffBanks is a full-stack project built with Vue 3, Pinia, Node.js, Express, Socket.io, MongoDB, Docker, and Nginx.

It is useful in this portfolio because it shows that I can build full applications, not only backend components or isolated experiments.
