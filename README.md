# Lorenzo Leoni Portfolio

Personal portfolio built with Vue 3, Vite, and Markdown content.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Content

- Project metadata lives in `src/data/projects.ts`.
- Project writeups live in `src/content/projects/`.
- Lab note metadata lives in `src/data/notes.ts`.
- Lab note writeups live in `src/content/notes/`.

The site uses hash-based routing so it works on GitHub Pages without server rewrites.
